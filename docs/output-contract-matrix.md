  type: `error`
  fields: none
  source: `_mutation_removed`
  notes: No success payload. Command always exits 2 because monday removed the mutation.
- `validation delete`
  type: `error`
  fields: none
  source: `_mutation_removed`
  notes: No success payload. Command always exits 2 because monday removed the mutation.
- `validation list`
  type: `object`
  fields: `required_column_ids`, `rules`
  source: `VALIDATIONS_LIST.validations`
- `validation update`
  type: `error`
  fields: none
  source: `_mutation_removed`
  notes: No success payload. Command always exits 2 because monday removed the mutation.

## webhook

- `webhook create`
  type: `object`
  fields: `id`, `board_id`, `event`
  source: `WEBHOOK_CREATE.create_webhook`
- `webhook delete`
  type: `object`
  fields: `id`, `board_id`
  source: `WEBHOOK_DELETE.delete_webhook`
- `webhook list`
  type: `list[object]`
  fields: `id`, `board_id`, `event`, `config`
  source: `WEBHOOKS_LIST.webhooks`

## workspace

- `workspace add-team`
  type: `list[object]`
  fields: `id`, `name`
  source: `WORKSPACE_ADD_TEAMS.add_teams_to_workspace`
- `workspace add-user`
  type: `list[object]`
  fields: `id`, `name`
  source: `WORKSPACE_ADD_USERS.add_users_to_workspace`
- `workspace create`
  type: `object`
  fields: `id`, `name`, `kind`, `description`, `state`
  source: `WORKSPACE_CREATE.create_workspace`
- `workspace delete`
  type: `object`
  fields: `id`
  source: `WORKSPACE_DELETE.delete_workspace`
- `workspace get`
  type: `object`
  fields: `id`, `name`, `kind`, `description`, `state`, `created_at`
  source: `WORKSPACE_GET.workspaces[0]`
- `workspace list`
  type: `list[object]`
  fields: `id`, `name`, `kind`, `description`, `state`, `created_at`, `_fuzzy_score?`
  source: `WORKSPACES_LIST_PAGE.workspaces[]`
  notes: When --fuzzy-score is used, _fuzzy_score is appended last.
- `workspace remove-team`
  type: `list[object]`
  fields: `id`, `name`
  source: `WORKSPACE_REMOVE_TEAMS.delete_teams_from_workspace`
- `workspace remove-user`
  type: `list[object]`
  fields: `id`, `name`
  source: `WORKSPACE_REMOVE_USERS.delete_users_from_workspace`
- `workspace update`
  type: `object`
  fields: `id`, `name`, `kind`, `description`
  source: `WORKSPACE_UPDATE.update_workspace`
